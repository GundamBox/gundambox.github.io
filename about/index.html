關於我

施華桐，自稱鋼彈，現在想想高中取這名字頗中二的。高中時候發現自己跟程式有緣，便以此為志向。

就讀大學期間，會回到高中的電腦研究社教導學弟妹寫程式，不敢自稱高手，但自認寫程式很龜毛。

C#、Python、C/C++、Java、PHP 都寫過，但都沒有到大型專案的程度。

貫徹什麼都碰，什麼都會，什麼都不會的魯蛇生活(大誤)

目前主要使用 ASP.NET Core 開發後端，配上 Azure DevOps 部署更是妙不可言(誤)

  • 2015-10-12 Create:

    以往高中社團的程式設計都是靠學長姊教導，鮮少紙本或電子資料流傳，教學也是隨性的教，於是就有了整理資料的念頭。而最近幾屆都是自己憑著熱血回饋社團，擔心自己開始就業之後會沒有人教導電腦研究社的學弟妹寫程式，所以憑著一股熱血想將自己的經驗都記錄下來，也順便累積屬於自己的筆記，於是本站就誕生了。
