github+hexo架設心得

發表於 2015-10-05 更新於 2024-03-28 分類於 Hexo Disqus：
文章字數： 664 所需閱讀時間 ≈ 1 分鐘
這篇是紀錄我如何架設blog的過程

其實很早就想將自己的學習心得放在網路上給高中的學弟妹看，不然大學畢業後就沒空教學弟妹了

這篇是紀錄我如何架設blog的過程，除了跟風(github.io的域名)，也是要給學弟妹的第一份筆記

一、本機端安裝

首先要先安裝node.js

最新版的node.js會自動加入環境變數，所以環境變數這邊就省了

接下來打開可愛的小黑窗安裝hexo

npm install -g hexo

這樣hexo就安裝完成了

二、github設定

到github開新的repository，命名為：用戶名.github.io

三、Hexo設定

建立新的資料夾並且初始化

mkdir hexo
hexo init

產生頁面

hexo generate

啟動server(預設為localhost:4000)

hexo server

可以使用**-p**來調整port

hexo server -p 8080

四、部署到github上面

修改hexo資料夾底下的_config.yml

到最底下修改

deploy:
    type: git
    repository: git@github.com:github帳號/repository名稱.github.io.git
    branch: master

記得要先建立SSH的Key喔，可以參考官方文件

最後輸入

hexo clean
hexo generate
hexo deploy

接著到http://{github帳號}.github.io就可以看到成果囉

心得

Well，用Markdown寫文章寫起來真的很爽，之後來找插件加強網站的功能囉
